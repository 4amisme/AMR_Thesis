## Installation Manual

### 1. Install Python

Please install Python version 3.x from the official Python website.

ตรวจสอบเวอร์ชัน Python:

```bash
python --version
```

หรือ

```bash
python3 --version
```

---

### 2. Create Virtual Environment

สำหรับ Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

สำหรับ macOS/Linux:

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 3. Install Required Libraries

ติดตั้ง library ที่จำเป็นจากไฟล์ `requirements.txt`

```bash
pip install -r requirements.txt
```

---

### 4. Prepare Dataset / Database

ถ้าโปรเจคมี dataset ให้ใส่ไฟล์ dataset ไว้ในโฟลเดอร์:

```text
dataset/
```

ถ้าโปรเจคใช้ database ให้ทำตามขั้นตอนดังนี้:

1. สร้าง database ชื่อ `database_name`
2. Import ไฟล์ SQL จากโฟลเดอร์ `database/`
3. แก้ไขค่าการเชื่อมต่อ database ในไฟล์ config เช่น username, password, host และ database name

## How to Run the Project

รันโปรแกรมด้วยคำสั่ง:

```bash
python main.py
```

หรือถ้าเป็นเว็บแอป:

```bash
python app.py
```

จากนั้นเปิดเว็บเบราว์เซอร์และเข้าไปที่:

```text
http://localhost:5000
```
